# AI Resume–Job Matching System (RAG-based)

A Retrieval-Augmented Generation (RAG) system that helps job seekers evaluate how well their resume matches job descriptions and identify missing skills.

---

## Features

- Job description ingestion
- Text chunking pipeline
- Semantic search using embeddings
- Resume-to-job matching
- Skill extraction
- Missing skill detection

---

## Tech Stack

- Python
- Sentence Transformers
- FAISS (vector search)
- LangChain (text processing)

---

## Architecture

Resume → Embeddings → Vector DB → Similarity Match → Skill Analysis

---

## Example Output

- Match Score: 0.53
- Missing Skills: PyTorch, TensorFlow
